## 我的IDEA常用快捷鍵及其工具
alt + E/B 光标跳转到段落手尾   
Ctrl + Shift + n 快速打开到某个窗口  
Alt + D 最大化代码窗口  
Setter and Getter Generate  
Ctrl + Alt + v 快速new 匿名对象为非匿名  
Ctrl + E 切换到最近编辑的窗口
右键+Generate+Setter Getter
Ctrl+Shift+Alt+j 修改全部名称
